# DrawFate

**จำลองการจับใบดำใบแดง** สำหรับกลุ่ม งานกิจกรรม หรือเกมปาร์ตี้ — ตั้งจำนวนใบ ดูโอกาส สุ่มผลได้เลย ไม่ต้องสมัครสมาชิก

🔗 **[เล่นได้เลย → jih9x.github.io/DrawFate-](https://jih9x.github.io/DrawFate-/)**

---

## ความหมายของใบ

| ใบ | ความหมาย | ผล |
|---|---|---|
| ใบดำ | รอดแล้ว! | การ์ดดำ + confetti |
| ใบแดง | โดน! | การ์ดแดง + สั่น |

---

## ฟีเจอร์

- **ตั้งจำนวนใบได้อิสระ** — ปุ่ม +/− แยกใบดำและใบแดง สูงสุด 99 ใบต่อประเภท
- **แสดงผู้เล่นที่ต้องการ** — ตัวเลขใหญ่อ่านง่าย อัปเดตตามที่ตั้ง
- **แสดงโอกาสโดน (%)** — คำนวณ live ทั้งก่อนและระหว่างการจับ
- **Animation การ์ดพลิก** — กดจับ → การ์ดพลิกเปิดผลทีละคน
- **ประวัติการจับ** — chip สีแสดงลำดับผลทุกรอบ
- **สรุปผลจบรอบ** — ยอดรอด/โดนเมื่อครบทุกคน
- **รีเซ็ตได้ทันที** — กลับสู่หน้าตั้งค่าโดยไม่โหลดหน้าใหม่

---

## วิธีใช้งาน

1. **ตั้งจำนวนใบ** — กด `+` / `−` กำหนดใบดำ (รอด) และใบแดง (โดน)
2. **ดูสถิติ** — ตรวจสอบจำนวนผู้เล่นและโอกาสโดน (%) ก่อนเริ่ม
3. **กด "จับใบ"** — ส่งมือถือให้แต่ละคนกด ผลเปิดเผยทีละคน
4. **จนครบทุกคน** — ดูสรุปผล แล้วกด รีเซ็ต เพื่อเริ่มรอบใหม่
