# 🎴 DrawFate

**จำลองการจับใบดำใบแดง** สำหรับกลุ่ม งานกิจกรรม หรือเกมปาร์ตี้ — ตั้งจำนวนใบ ดูโอกาส สุ่มผลได้เลย ไม่ต้องสมัครสมาชิก

---

## ✨ ฟีเจอร์

- **ตั้งจำนวนใบได้อิสระ** — ปุ่ม +/− แยกใบดำและใบแดง สูงสุด 99 ใบต่อประเภท
- **แสดงจำนวนผู้เล่นที่ต้องการ** — ตัวเลขใหญ่อ่านง่าย เห็นชัดทันที
- **แสดงโอกาส (%)** — คำนวณ live ทั้งก่อนและระหว่างการจับ
- **Animation การ์ดพลิก** — กดจับ → การ์ดพลิกเปิดผล
  - ใบดำ: แจ้งเตือนพร้อม effect สั่น
  - ใบแดง: แสดงผลพร้อม confetti
- **ประวัติการจับ** — แสดง dot สีทุกรอบตามลำดับ
- **สรุปผลจบรอบ** — แสดงยอดใบดำ/ใบแดงเมื่อครบทุกคน
- **รีเซ็ตได้ทันที** — กลับสู่หน้าตั้งค่าโดยไม่ต้องโหลดหน้าใหม่

---

## 🚀 วิธีใช้งาน

1. **ตั้งจำนวนใบ** — กด `+` / `−` เพื่อกำหนดใบดำ (โดน) และใบแดง (ปลอดภัย)
2. **ดูสถิติ** — ตรวจสอบจำนวนผู้เล่นที่ต้องการและโอกาส (%) ก่อนเริ่ม
3. **กด "จับใบ"** — ส่งให้แต่ละคนกด ผลจะเปิดเผยทีละคน
4. **จนครบทุกคน** — ดูสรุปผล แล้วกด รีเซ็ต เพื่อเริ่มรอบใหม่

---

## 🛠 เทคโนโลยี

- HTML + CSS + JavaScript ล้วน (ไฟล์เดียว ไม่มี framework หรือ build step)
- ฟอนต์ [Sarabun](https://fonts.google.com/specimen/Sarabun) จาก Google Fonts
- รองรับมือถือ (mobile-first responsive)

## 📦 การ Deploy บน GitHub Pages

1. ไปที่ **Settings → Pages** ใน repository นี้
2. เลือก **Branch: `main`** / **Folder: `/ (root)`**
3. กด Save — เว็บจะพร้อมใช้ที่ `https://<username>.github.io/DrawFate-/`
